## OpenSCAD Heart Coaster

I used the following prompt to create an OpenSCAD script that generates a repeating 
pattern on a coaster.  The coaster also has a label for a name or other word.

[Prompts.Developer.OpenScad.REPEATING_PATTERN_COASTER](../../../images/prompts/src/main/java/org/onebeartoe/prompts/Prompts.java)

The prompt mentions the source of the original script by Mike Kasberg.
