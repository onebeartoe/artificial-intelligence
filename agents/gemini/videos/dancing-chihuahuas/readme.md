# Gemini Veo - Dancing Chihuahuas

A family member has two Chihuahuas that seen in the photo in this directory.

I wanted to see if could get Veo to create a silly video of them dancing.

At first, I wanted to use JBang to interact with Veo.  The main block was me not knowing how to reuse the prompt in another Java file.  It ended up required a build, include the JAR on the classpath and then use it in the Java file in this directory.

But I ran out of time (for this part) and just used the prompt with Gemini CLI.

The [resulting video](dancing-chihuahuas-hands-from-start-h264-crf23.mp4) is also in this directory.
