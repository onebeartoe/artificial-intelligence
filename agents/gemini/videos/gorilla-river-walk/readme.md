# Gorilla on the Riverwalk! (CLI)

Once I was convinced that Gemini Veo is capable of making silly short videos, I wanted to 
see how it would do on the command line interface (Gemini CLI)

I am from San Antonio, TX and though it would be extra silly to see a gorilla in a bot on the River Walk!

Some were even scary.

So I added 'nice' and 'not intimidating' to the video prompt.

This was using Gemini CLI, but the actual prompt is stored in [GORILLA_ON_THE_RIVER_WALK](../../images/prompts/src/main/java/org/onebeartoe/prompts/Prompts.java) to create the gorrilla on the San Antonio River Walk in a boat.


Note: this is where it started costing money.  It was $28 for 8 videos I only like two though.

See the video is this directory for the prompt output.
