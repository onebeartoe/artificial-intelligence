# How Silly Can You Get? - Video Loop with Music - Gemini Veo

This video combines all silly outputs I have made so far with Gemini; images, 'music' and video.

It loops the video created from an image and plays a 'song' on loop over the video.

The uber-silly video, [compressed-looped_video_with_music-2026-04-09.mp4](compressed-looped_video_with_music-2026-04-09.mp4) is in this same directory.

Hopefully you find it silly too!

And again it was inspired by the surfing scene in the movie 'Top Secret'.  R.I.P. Val Kilmer.

The code, [BackgroundMusicToLoop.java](src/main/java/org/onebeartoe/ai/background/music/to/loop/BackgroundMusicToLoop.java), is under src/ in this directory.
